# 投票アプリ

## アプリケーション概要
ヒヤリハットアプリは、様々な場面で発生するヒヤリハットを報告・確認することができるアプリケーションです。
このアプリケーションはモバイルでの利用を想定しており、ユーザーはどこでも簡単に報告・確認を行えます。
また、各種ランキング表示やグラフ表示などが備わっており発生状況を把握することに役立ちます。
