# AI ヒヤリハットアプリ

## アプリケーション概要
AIヒヤリハット報告アプリは、生成AIを活用し、ヒヤリハット登録時の支援を行い、過去のヒヤリハットを元に注意すべきポイントの提案を行うアプリです。
過去に登録したヒヤリハットの情報を基に回答を生成するため、自社に適した提案を受けることができます。
